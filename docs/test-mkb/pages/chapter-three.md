# 第三章 画像つきの章

この章では、`assets/` 内に置かれた画像が MD 内の相対パス参照で正しく Blob URL に置換されることを確認します。

![テスト画像](assets/test-image.png)

画像のあとにも文章が続くことで、画像とテキストの混在レイアウトを検証します。CSS multi-column を使ったページネーションでは、画像がページ境界をまたぐと崩れることがあります。これを防ぐために `break-inside: avoid` を画像に適用しています。
