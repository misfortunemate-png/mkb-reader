# mkb-reader 検証用サンプル

このファイルは **mkb-reader** の §1〜§4 を実機で検証するためのサンプル MarkBook です。

## このサンプルに含まれる要素

1. 日本語の長文（小説風の散文）
2. 技術文書（コードブロック、テーブル）
3. 画像参照を含む文書

### 検証ポイント

- ページネーションの操作感
- 日本語明朝体での読書感
- 画像・コードブロックがページ境界をまたぐ際の挙動

> 「美しく読む」を目的にした以上、ここで揺らげば後ろが揺らぐ。だから検証はここから始まる。

#### 構成

- [[chapter-one]] — 長文（小説風）
- [[chapter-two]] — 技術文書
- [[chapter-three]] — 画像つき

##### 補足

wikilinks（`[[...]]`）でチャプター間を移動できることも、ここで確かめる。

###### 注

存在しないチャプターへのリンクは [[ghost-chapter]] のようにグレーアウトされる予定。
